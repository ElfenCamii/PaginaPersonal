Copia de pagina web
de Tesla

Descripción

Pagina Web desarrollada con HTML, CSS, Astro y Tailwind. Estos dos ultimos son herramientas de construcción de sitios web estáticos y aplicaciones web, el primero de estos se enfoca en la velocidad y el rendimiento ya que compila componentes web en el servidor y entregar solo el código necesario. Mientras que el segundo es un framework de CSS que proporciona una serie de clases predefinidas que se pueden aplicar directamente en el HTML
